# Changelog

All notable changes to this project will be noted here.

## Version 1

### [1.0.2] - 26 May 2025

Refactored the code for maintainability, reusability and ease of expansion by making the use of object oriented programming.

### [1.0.1] - 15 May 2025

This maintenance release resolves two UI bugs: it fixes an issue where the space bar was unresponsive within content input fields, and corrects a bug preventing the 'Today' button from functioning properly when navigating calendar views via search results.

### [1.0.0] - 11 Aug 2023

Initial release of the Online Diary application. This version introduces a dynamic, calendar-based interface for organizing daily journals, secure client-side storage utilizing `localStorage`, and a customizable settings panel for personalization (theme and time format configurations). It also includes a title-based entry search engine with typewriter placeholder animations and a write-lock mechanism to preserve historical entries as read-only.
